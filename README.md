# switchable

> Create amount of daily plans, then quickly switch from one to another. 

#### Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:9080, d=dev
yarn run d

# build electron application for production, m=mac, l=linux, w=win32
yarn run build
yarn run m
yarn run w
yarn run l
yarn run build:web


```
